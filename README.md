# Route Recommendation App

A web application that provides route recommendations with weather advisories and location-based services.

## Features
- Location-based route planning
- Weather advisory integration
- Real-time traffic updates
- Route optimization algorithms
- Responsive UI for all devices

## Technologies Used
- Frontend: React TypeScript with Mapbox GL JS
- Backend: Python Flask with ML models
- Dependencies: Mapbox API, OpenWeatherMap API

## Getting Started
1. Clone the repository
2. Install dependencies:
   ```bash
   cd backend && pip install -r requirements.txt
   cd frontend && npm install
   ```
3. Set up API keys in `.env` files
4. Run the application:
   ```bash
   cd backend && python main.py
   cd frontend && npm run dev
